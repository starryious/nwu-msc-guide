# 关于数学分析

## 课程简介

数学分析，作为国内大学数学系最为基础的数学课程之一，要学习三个学期，王lt教授曾对我们说，数学分析是相对较难自学的一门课程，而且大概高强度学习三年才能差不多吃透数学分析，把这么多的内容在三个学期中教完，学生的理解程度是肯定不够的。因此我们需要十分认真的学习数学分析这门课程，若是打好分析的基础，无论是之后走纯数学的各个方向，还是走应用数学都会事半功倍

## 课程内容学习

大一的数院新生刚进来之后，拔尖班的数学分析课程会由王lt教授上，王lt教授教授的数学分析课的质量是非常高的，至少在我大一这一年中我认为是质量最高的一门课程，在学习数分新课之前一定是要预习的，预习十分关键，十分关键，十分关键！接着我认为听课相对而言没有预习这么关键，王lt上课的内容和例题主要都是来自陈纪修和徐森林（可能稍微夹一点菲尔金哥尔茨和一些其他教材里的内容），大部分课上的内容信息密度都极大，但如果你预习了那一般来说没什么问题，接着课后复习也是十分关键的，想学的十分透彻的话课后习题可以全部做完，当然这很花时间，部分计算类的东西可以少做几道，当然一定是在做会的前提下少做一点（很少计算的话计算能力会下降的十分厉害）

## 书籍推荐

在这个库中已经放了一些书籍的推荐的文件，其中一个是清华大学的一个数学分析书单推荐，当然里面的教材的难度都很高。
我建议初学者可以从徐森林（十分详尽，例题众多，习题思考题质量高且有配套答案），张筑生（内容较新，但没有习题），梅加强（证明简洁，有作者的自己独到的观点），这三本中挑一本作为精读的书籍，陈纪修在有部分内容上写的没有那么让人容易理解。
